this is hex-tac-toe. this answers the question: can tic tac toe, with hexagons?

Controls: q,a,z,e,d,c for movement, space/enter for placing. 

Rules:
- Infinite hexagonal grid
- X goes first, once
- Each player following goes twice in a row (so O goes twice, X goes twice, etc).
- First to get six in a row wins
Have fun!

How to run: Run make.
Set up the server by running ./server. It will tell you in the console what IP address and port to connect to.
Then, run ./client with the IP address and port you see

If you want to play offline, just run ./hexagon.
